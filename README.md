# Progetto_18_Agosto_Gabriele_Daniele_Miriana
Repository per il progetto pomeridiano di gruppo del 18 agosto, corso AI Academy.

Programma a menù testuale che prevede le seguenti operazioni alla pressione dei numeri.

1. aggiungi_studente(nome, cognome, lista): inserimento di nome e cognome dello studente nella lista specificata.
2. elenca_studenti(lista_studenti): mostra tutti gli studenti in ordine alfabetico, la ricerca viene eseguita principalmente sul cognome con un check secondario sul nome in caso di doppione.
3. cerca_studente_per_nome(nome, lista_studenti): cerca studenti per nome tramite match parziale della lista. In particolare in base al nome inserito verrà mostrata una lista di nomi e cognomi compatibili.
4. termina_programma(): chiusura del programma e della console.
